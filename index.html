Medicaid Kenya

Powered by Esri maps, Medicaid Kenya is an android application coded in Java using firebase for hosting its database. It is a platform where the ambulance or any medical emergency respose service can connect with patients in dire need of medical assistance. The app displays an estimated time of arrival for the emergency response team heading to a patients pickup location

Maji Nuni

Maji Nuni was a project that I worked on with my team in IBM during a one month period. It used a mapping tool and analytics to measure and locate bore holes and other fresh water sources in mandera county. We used IBM bluemix to access the data.
Requirements: Docker, Docker
Built with: Bootstrap, Javascript Leaflet JS for mapping and Python Django.
